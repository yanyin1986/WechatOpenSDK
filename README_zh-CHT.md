# WechatOpenSDK

[English](README.md) | [简体中文](README_zh-CHS.md) | 繁体中文

## 目的

- 包裝 `WeChatOpenSDK.xcframework` 讓其可以簡單得用於 `Swift Package Manager`
- 保持與Cocoapods的版本相同
