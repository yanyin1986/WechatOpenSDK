# WechatOpenSDK

[English](README.md) | 简体中文 | [繁体中文](README_zh-CHT.md)

## 目的

- 包装 `WeChatOpenSDK.xcframework` 让其可以简单得用于 `Swift Package Manager`
- 保持与Cocoapods的版本相同
